# 计数原理知识清单（高二文科）

## 一、两个基本计数原理

| 原理 | 关键词 | 逻辑关系 | 运算 | 适用情形 |
|------|--------|----------|------|----------|
| 分类加法 | 分类、加法 | 或（任一类别完成即可） | 加法 \(N = m_1 + \dots + m_k\) | 互斥的几类方案 |
| 分步乘法 | 分步、乘法 | 且（所有步骤缺一不可） | 乘法 \(N = m_1 \times \dots \times m_k\) | 依次完成的独立步骤 |

> **易错提醒**：加法原理要求各类**互斥**（不重复）；乘法原理要求各步**独立**（方法数不受之前步骤影响）。

## 二、排列与组合

| 概念 | 定义 | 顺序是否重要 | 符号 | 公式 | 关系 |
|------|------|--------------|------|------|------|
| 排列 | 从 \(n\) 个不同元素中取 \(m\) 个排成一列 | **重要** | \(A_n^m\) 或 \(P_n^m\) | \(A_n^m = n(n-1)\cdots(n-m+1) = \frac{n!}{(n-m)!}\) | \(A_n^m = C_n^m \times m!\) |
| 组合 | 从 \(n\) 个不同元素中取 \(m\) 个作为一组 | **不重要** | \(C_n^m\) 或 \(\binom{n}{m}\) | \(C_n^m = \frac{A_n^m}{m!} = \frac{n!}{m!(n-m)!}\) | \(C_n^m = \frac{A_n^m}{m!}\) |

**特殊规定**：  
- \(0! = 1\)，\(A_n^0 = 1\)，\(C_n^0 = 1\)  
- \(A_n^n = n!\)，\(C_n^n = 1\)  
- 组合数性质：\(C_n^m = C_n^{n-m}\)，\(C_n^m = C_{n-1}^{m-1} + C_{n-1}^{m}\)（杨辉三角）

## 三、常用符号体系

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| \(n!\) | \(n\) 的阶乘 | \(n \times (n-1) \times \cdots \times 2 \times 1\) | \(5! = 120\) |
| \(A_n^m\) | 排列数 | 从 \(n\) 个不同元素中取 \(m\) 个排列的种数 | \(A_5^2 = 20\) |
| \(C_n^m\) | 组合数 | 从 \(n\) 个不同元素中取 \(m\) 个组合的种数 | \(C_5^2 = 10\) |
| \(\sum\) | 求和符号 | 表示若干项的和 | \(\sum_{i=1}^{k} a_i = a_1+\cdots+a_k\) |

## 四、重要数学思想

1. **分类讨论**：将复杂问题划分为互斥的几类，分别计数再相加。  
   → 关键：**不重不漏**。

2. **有序 vs 无序**：  
   - 题目出现“排成一排”“安排职务”“编号不同” → 排列（顺序重要）。  
   - 题目出现“选出一组”“抽取样品”“组成集合” → 组合（顺序无关）。

3. **化归思想**：将陌生问题转化为基本模型（加法、乘法、排列、组合）。  
   - 常用技巧：**先选后排**（先组合再排列）、**特殊元素优先处理**、**相邻问题用捆绑法**、**不相邻问题用插空法**。

4. **正难则反**：用总数减去不符合条件的情况（补集法）。

## 五、常见易错点与提醒

| 易错情形 | 正确做法 |
|----------|----------|
| 混淆加法与乘法 | 问自己：“完成目标需要同时满足几个条件？”是→乘法；“有几种不同的独立途径？”是→加法。 |
| 排列组合误用 | 判断顺序是否改变结果：交换两个元素后结果不同 → 排列；相同 → 组合。 |
| 数字组数问题遗漏0的限制 | 首位不能为0，需要单独考虑。 |
| 重复计数或遗漏 | 使用分类时检查各类是否互斥；使用分步时检查每步方法数是否恒定。 |
| “至少”“至多”问题 | 常用补集法，或直接分类（注意不重不漏）。 |
| 组合数公式中忘记除以 \(m!\) | 牢记 \(C_n^m = A_n^m / m!\) 来源于“排列中每个组合被重复计算了 \(m!\) 次”。 |

## 六、典型问题类型快速索引

| 问题类型 | 主要方法 | 示例 |
|----------|----------|------|
| 组数问题（数字） | 优先考虑特殊位（首位、个位），分步或分类 | 用0~4组成无重复数字的四位偶数 |
| 排队问题 | 排列，注意特殊位置、相邻（捆绑）、不相邻（插空） | 女生不相邻的排法 |
| 选人问题 | 组合，注意是否带职务（有职务则用排列） | 选3人参加座谈会（组合）；选班长和副班长（排列） |
| 分配问题 | 先选后排，或直接分步 | 将4本书分给3人 |
| 几何计数 | 组合（选点、选线） | 12点中任取3点构成三角形 |
| 涂色问题 | 分步乘法，注意相邻区域颜色不同 | 用m种颜色给地图区域涂色 |
