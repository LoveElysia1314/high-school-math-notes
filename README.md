# 导数入门与应用

这是一个给高中学生准备的导数学习资料仓库，主要内容都用 Markdown 编写，方便直接在浏览器里阅读。

## 怎么在线阅读

### 方式一：在 GitHub 上看

1. 打开这个仓库的 GitHub 页面。
2. 进入 `导数入门` 文件夹。
3. 直接点击 `.md` 文件阅读，例如正文、习题和知识清单。
4. 如果你看到的是源码文本，就点页面上的预览或返回仓库页再打开文件。

### 方式二：通过 GitClone 镜像看

1. 打开仓库对应的 GitClone 页面。
2. 找到 `导数入门` 文件夹。
3. 点击 `.md` 文件即可在线阅读。

1. [导数入门_正文.md](导数入门_正文.md)

## 小提示

- Markdown 文件以 `.md` 结尾，适合在网页里直接看。
- 公式通常会用 `$...$` 或 `$$...$$` 表示。
- 如果网络条件允许，优先在线阅读；如果需要打印或离线看，再下载文件。

## 这个仓库里有什么

- `导数入门_正文.md`：导数基础内容
- `导数应用_正文.md`：导数应用内容
- `导数入门_习题与解答.md`：基础练习和答案
- `导数应用_习题与解答.md`：应用练习和答案
- `知识清单.md`：核心知识点总结

## PDF 收集脚本

如果需要把某个文件夹里的 PDF 递归提取并放到根目录，可以运行仓库根目录下的 [collect_pdfs_to_root.py](collect_pdfs_to_root.py)。

示例：

```bash
python collect_pdfs_to_root.py 导数入门 --output-dir pdf
```

上面的命令会把 `导数入门` 下的所有 PDF 扁平化复制到 `pdf` 目录；如果要直接移动文件，可以加 `--move`。也可以通过 `--archive 包名.zip` 再生成一个压缩包。
