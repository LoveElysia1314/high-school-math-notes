## 提示词：Markdown 转教学风格 LaTeX 文档

### 任务
将给定的 Markdown 文稿转换为 LaTeX 源码，并填充到提供的模板的 `\begin{document}` 与 `\end{document}` 之间。

### 输入
- Markdown 文稿（学习笔记，包含文本、公式、列表等）
- LaTeX 模板（包含所有必要宏包和页面设置，无需修改导言区）

### 输出
一个完整的 `.tex` 文件，模板导言区保持不变，仅替换文档主体内容。

### 转换要求

1. **语言风格**：专业严谨，和蔼但不刻意追求生动。重要结论用 `\begin{quote}\textbf{结论：} ... \end{quote}` 突出。

2. **章节标题**：一级 `\section{}`，二级 `\subsection{}`……，去除原稿中的编号，交由 LaTeX 自动生成。

3. **数学公式**：行内 `$...$`，独立 `$$ ...$$`。长公式优先使用 `align*` 环境对齐等号。

4. **段落与列表**：
   - 段落之间必须用空行分隔。
   - 通常将列表内容改写为自然段论述，使行文流畅，而非大量使用 `itemize`、`enumerate` 等列表环境，但这绝不意味着可以用手动打点或序号代替列表环境。当信息最适合使用列表环境时完全可以使用。
   - 避免使用 `\textbf{小标题：}` 来分割内容，应将其融入自然段落或通过上下文过渡。

5. **符号一致性**：同一概念在全文中使用相同符号，不得混用。

6. **其他**：表格用 `tabular` 简单呈现；忽略图片转换；保持 LaTeX 源可读性。

### 模板示例
```latex
\documentclass[10pt]{article}
\usepackage[UTF8]{ctex}
\usepackage{amsmath, amssymb}
\usepackage{geometry}
\geometry{a4paper, margin=1.5cm}
\usepackage{enumitem}

\title{标题放这里}
\author{留空}
\date{留空}

\begin{document}
\maketitle
% 转换后的内容放在这里
\end{document}
```

请根据以上要求输出完整的 `.tex` 文件。
